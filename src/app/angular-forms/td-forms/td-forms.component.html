Template driven forms

Steps:

  • Import the FormsModule module inside the container module (generally, app.module.ts)
  • Add the template reference name to the form tag and set its value to ngForm. ie: #tRefName="ngForm"
  • Add the (ngSubmit) event listener to the form tag and the event handler function to your controller in one of below ways:
    • (ngSubmit)="submit(#tRefName)" and adding to controller submit(myForm: NgForm) {{ '{' }}...{{ '}' }} or
    • (ngSubmit)="submit()" and add the ElementRef to the controller @ViewChild('tRefName') myForm: NgForm; and submit() {{ '{' }}...{{ '}' }}
  • Add the name attribute with their respective value to all of your form controls
  • Add the ngModel directive to all inputs in your form.
    to get the reference to the form control, especially for validations, use: #refName="ngModel"
  • to set default values to a form control, use: [ngModel]="propertyName"
    If you want double way binding add: [(ngModel)]="propertyName"
  • Add the HTML validations Angular Validators class
    Additionally, you might also want to enable HTML5 validation (by default, Angular disables it). You can do so by adding the ngNativeValidate to a control in your template.
  • Otional: You can make form groups using: ngModelGroup="myGroup" #myGroup="ngModelGroup" in the group wrapper element
